# Searchy

PHP Search Engine

## Features

1. Basic Search Engine Features
2. Simple Crawler
3. Add Site Feature

## How to use

1. `$ composer update`
2. Update crawler/urls.txt with the urls you want to index like Alexa top 1000
3. Update config.php
4. Import search.sql to your database
5. Run `$ php crawler.php `
6. Enjoy
